# FridgeFinder 🍽️

En receptapp där du kan hitta recept baserat på ingredienser du redan har hemma.

## Tekniker

- React Native
- Expo Router
- TypeScript
- Spoonacular API

## Funktioner

- Startsida med slumpmässiga receptförslag
- Sök recept på ingredienser
- Detaljvy för varje recept

## Starta appen

1. Klona projektet
2. Kör `npm install`
3. Kör `npx expo start`
